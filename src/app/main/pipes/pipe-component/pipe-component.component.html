Pipes:

Pipes are simple functions to use in template expressions to accept an input value and return a transformed value. Pipes are useful because you can use them throughout your application, while only declaring each pipe once.

Pure Pipe

A pure pipe is called only when Angular detects changes in the input reference. This means that a pure pipe will not re-run if only the content of an array or object changes; the pipe will re-run only if the array or object itself is replaced with a new instance.

Example Use Cases: Capitalizing text, formatting dates, converting numbers.

Impure Pipe

An impure pipe is called on every change detection cycle, meaning it will re-run even if there is no change in the input reference (like when a property inside an array or object changes).

Example Use Cases: Filtering a list of items, live data transformations where frequent updates occur.
