String Interpolation

String Interpolation is a one-way databinding technique which is used to output the data from a TypeScript code to HTML template (view)

Button remain disabled even when the value is set to false using string interpolation as the each value is considered as string.

              
                <button  disabled= "{{isDisabled}}"> Disabled </button>
{{item}}
In the above example if the user inputs < b> Text < /b>

the text appears normal

Property Binding

In Angular, property binding is a one-way data binding mechanism that allows you to set the properties of HTML elements or Angular directives dynamically.

Button is Enabled using property binding and works fine.

              
                <button  [disabled] = "isDisabled"> Disabled </button>
In the above example if the user inputs < b> Text < /b>

the text appears in bold text

A user can manipulate the dom which is not ideal
